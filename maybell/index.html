Tailwind - E-commerce

A collection of free E-commerce website page templates, built with the utility-first Tailwind CSS framework. MIT licensed - free for personal and commercial use.

Getting started

Clone the project
git clone https://github.com/bbulakh/tailwind-ecommerce.git
Done!
Install the dependencies:
npm install
Dependencies have been installed succesfully!
Run the project
Start server
npm run start
Or Build
npm run build
💜 Use it for your purposes and share your feedback with me!
